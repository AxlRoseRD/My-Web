# My-Web
Practica Electiva Devops CI/CD
Crear una pagina web hola mundo que se despliegue automaticamente a un servicio hosting web en la nube desde un repositorio de github. Utilizar github pages  o cloudflare pagespara automatizar la creacion de la pagina con el contenido del repositorio de github.   Pegar aqui url del repositorio y de su pagina de github.io o Cloudflare.
